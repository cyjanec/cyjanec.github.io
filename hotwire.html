The Hot-Wire Technique of Determining Thermal Conductivity

Summer 2021 - As part of a summer research fellowship project, I collaborated with a teammate to adapt current transient hot-wire techniques to work with electrically-conducting materials at high temperatures of up to 1000 degrees Celcius.

LabVIEW Programming

Individually, I programmed a LabVIEW GUI-based data analysis application to calculate a material's thermal conductivity as datapoints are collected. LabVIEW is advantageous because it allows data collection and calculation to take place simultaneously. Also specific to the calculation of thermal conductivity is graphing data in a semi-logarithmic scale, and obtaining the slope of the data's linear region. The application provides cursors for the user to manually select the optimum linear range to use in calculations.
